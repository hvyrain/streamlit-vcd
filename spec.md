아래 지침에 따라 5개의 streamlit 파일을 생성한다.

조건
1. 디자인과 학생들을 위한 앱이니까 UI가 1. 아주 깔끔하고 세련되어야 해.
1. 가능한 이해가 쉬운 짧은 코드로 생성해.
1. 코드 주석에 각 부분이 어떤 역할을 하는지 중학생도 이해할 수 있게 한글로 설명해줘.

생성할 파일
1. 01_color_palette.py,컬러 팔레트,"""RGB 값을 조절하는 3개의 슬라이더를 만들고, 선택한 색상을 화면 중앙에 큰 박스로 보여줘. HEX 코드와 RGB 값을 텍스트로 표시하고 실시간 반영되게 해줘.""","슬라이더는 가로로 배치(st.columns), 컬러 박스는 둥근 모서리와 그림자 효과 적용."
2. 02_font_preview.py,폰트 미리보기,"""사용자가 텍스트를 입력하면 다양한 구글 폰트(Roboto, Playfair Display 등)가 적용된 샘플들을 나열해줘. 슬라이더로 글자 크기와 자간(Letter Spacing)을 조절하는 기능을 넣어줘.""",폰트별로 카드 형태의 레이아웃 구성. 대비가 확실한 타이포그래피 배치.
3. 03_filter.py, 지 필터,"""사용자가 JPG/PNG 파일을 업로드하면 흑백(Grayscale), 블러(Blur), 세피아(Sepia) 필터를 적용하고 원본과 비교해 보여줘. 결과물을 다운로드하는 버튼도 만들어줘.""",st.file_uploader 사용. Pillow 라이브러리 활용. Before/After 레이아웃 필수.
4. 04_data.py, 데이터 인포그래픽,"""간단한 숫자가 담긴 표(예: 디자인 트렌드 선호도)를 만들고, 이를 예쁜 막대그래프나 파이차트로 시각화해줘. 차트 색상은 디자인과 어울리는 파스텔 톤으로 설정해줘.""",st.bar_chart나 plotly 사용. 복잡한 표보다는 한눈에 들어오는 그래픽 강조.
5. 05_portfolio.py,AI 포트폴리오,"""디자인 작품 이미지들을 카테고리(로고, 웹, 영상)별로 필터링해서 보여주는 갤러리를 만들어줘. 각 이미지를 클릭하면 상세 설명이 나오는 모달이나 확장기 기능을 넣어줘.""",st.tabs나 st.selectbox로 필터 구현. 깔끔한 그리드 형태의 이미지 나열.
